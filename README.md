# archives67-urlFix
Supprime la limitation de taille dans l'url de téléchargement d'un document

Sur le nouveau site des Archives départementales d'Alsace (https://archives67.alsace.eu), il y a une limitation de taille de l'image lors de l'utilisation de bouton "Télécharger le document courant".

Cette extension à installer dans le navigateur supprime la limitation de la taille dans l'URL et permet de choisir si on souhaite télécharger le document ou le copier dans son presse papier.

Dès que le site ne comprendra plus cette limitation, cette extension n'aura pas d'utilité.
